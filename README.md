#WeChat AuthSDK Demo App

微信登录Demo为微信团队开源项目，用于微信开发者进行微信登录、分享功能开发时的参考Demo。微信开发者可以参考项目中的代码来开发应用，也可以直接使用项目中的代码到自己的App中。

开发者可以自由使用并传播本代码，但需要保留原作者信息。

联系我们：dev@wechat.com
